AKASHI CLI client written in Go

## Installation

```
go install github.com/hapoon/ak4
```

## Usage

使用する企業IDとアクセストークンの設定

```
ak4 init --login_company_code [企業ID] --token [アクセストークン]
```

打刻の実施

```
ak4 stamp
```
